# CV & Resume Portfolio Hub

A modern web application built with React, Vite, and Framer Motion that allows users to create elegant, minimal CVs, Resumes, and instant personal portfolio websites.

## Features
- Minimalist, professional form builder
- Live PDF Resume Preview
- Multiple templates (Modern, Minimal, Creative)
- Instantly generated animated personal Portfolio page
- One-click PDF download
- Clean UI with Framer Motion animations
